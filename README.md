# n8n Sri Lanka — Community Guidelines

Official Community Guidelines page for the **n8n Sri Lanka Community**.

Built as a premium single-page website with smooth animations, editorial layout, and a professional SaaS aesthetic.

## 🚀 Stack

- **Next.js 16** (App Router, Static Export)
- **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion** — scroll-triggered animations & micro-interactions
- **Lucide React** — icons
- **DM Sans** — typography (Google Fonts)

## ✨ Sections

1. **Hero** — Badge, headline, description, value badges, scroll indicator
2. **Community Guidelines** — 7 animated cards covering all community rules
3. **Moderation** — Consequence pills + moderation team callout quote
4. **Community Values** — 6 premium feature cards
5. **Footer** — Tagline, participation agreement

## 🏃 Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
